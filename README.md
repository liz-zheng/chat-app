## How to run app

1. In your terminal, cd to this repo
```bash
$ cd chat-app
```
2. Run the following commands
```bash
$ npm install
$ npm start
```
3. Open a new terminal and run
```bash
$ npm run api-server
```
4. Open a new terminal and run
```bash
$ npm run socket-server
```
5. Open http://localhost:3000/ in browser
